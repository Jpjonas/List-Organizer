Como fazer commit github (deixem que eu faco no final de cada dia):

1 - git init //nao precisa mais ser feito, é só para adicionar o arquivo de controle de versao do repositorio

2 - git add README.md //para adicionar arquivos no controle de versão

3 - git commit -m "Descricao do commit" //para adicionar o comentario do que foi feito na versão

4 - git remote add origin https://github.com/Jpjonas/Trabalho2PRA.git //para linkar com o repositorio na nuvem do github

5 - git push -u origin master //pede o email e senha para dar o commit para o nuvem
